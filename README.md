# 영화 예매 프로그램 (DBMS)
DataBase Movie Service with *C#*

using *Oracle Database*

## Details
- **데이터 베이스**와 연동하여 구현
- **기본 기능**: 회원가입, 아이디·비밀번호 찾기
- **관리자**: 회원가입 승인, 회원등급 관리, 등급 할인율 수정, 영화 등록, 상영시간표 등록
- **회원**: 회원정보 수정, 회원 탈퇴, 영화 예매, 예매 수정 및 취소
